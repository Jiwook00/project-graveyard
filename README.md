# 사이드프로젝트의 무덤 (Project Graveyard)

> _"여기 잠든 프로젝트들은 우리에게 소중한 가르침을 남기고 떠났습니다."_

완료되지 못한 프로젝트들의 이야기가 모이는 곳입니다. 중단된 사이드프로젝트들을 기리고 그로부터 얻은 교훈을 공유하는 플랫폼입니다.

## 기술 스택

- **Frontend Framework**: React 18
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **UI Components**:
  - Radix UI
  - Custom Card Components
- **Code Quality**:
  - ESLint
  - React Hooks Plugin

## 시작하기

### 필수 조건

- Node.js 18.0.0 이상
- npm 또는 yarn

### 의존성 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

### 빌드

```bash
npm run build
```

### 빌드 파일 실행

```bash
npm run preview
```
